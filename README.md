## Projeto de implementação da matéria de Fundamentos de Sistemas Operacionais

# Como executar

 - Precisa ter o compilador de cpp em sua máquina e executar o comando:

  ```
  make main
  ```
  Após isso é so executar o seguinte comando:

  ```
  ./main
  ```
